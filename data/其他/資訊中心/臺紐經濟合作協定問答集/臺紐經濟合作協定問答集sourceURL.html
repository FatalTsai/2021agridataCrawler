常見問答集(FAQ)

臺紐經濟合作協定問答集

問1 臺紐ANZTEC農業內容包括那些？
答
  1. 臺紐ANZTEC與農業有關的內容包括農產品市場開放、特定原產地規則及「食品安全檢驗與動植物防疫檢疫」（SPS）專章規範。
  2. 農產品市場開放：紐西蘭承諾在協定生效後，所有農產品關稅均降為零；我國則將稻米排除，另針對鹿茸與液態乳實施關稅配額，至於其他產品則分為立即取消關稅、2年後取消、3年後取消、4年後取消、6年後取消及8年後取消關稅等不同開放程度
  3. 特定原產地規則：為避免第三國經由轉運享受臺紐ANZTEC的優惠關稅，雙方已針對各項產品訂定可享受優惠關稅之原產地規則。 
  4. SPS專章：SPS專章條文共計16條，規範臺紐雙方遵守國際組織規定與相關科學證據之義務。主要內容包括建立｢執行協議｣機制，由雙方針對同等效力、驗證、發證、進口檢查等實施細節進行討論；明訂通知、諮商及資訊交換機制；以及成立聯合管理委員會等。
問2 我方承諾開放哪些農產品？有甚麼影響？
答
  1. 我國將稻米類產品排除在降稅清單外，並針對液態乳、鹿茸採取關稅配額，至於其他農產品則區分為立即取消關稅、2年後取消、3年後取消、4年後取消、6年後取消及8年後取消關稅等不同開放程度。 
  2. 鹿茸：配額內零關稅，配額量由協定生效後之1,000公斤，每年增加250公斤（第11年為3,750公斤），配額外關稅則由500%，每隔4年降100%，並自第12年起取消配額。
  3. 液態乳：配額內零關稅，配額量由協定生效後之5,500公噸，每隔3年增加1,500公噸（第9年起為1萬公噸），配額外關稅為每公斤14元，並自第12年起取消配額。
  4. 紐西蘭與臺灣分屬南北半球，農作物種類不同，預估影響將在畜產品方面。其中由於我國鹿茸多在養鹿場割取，另紐國增加液態乳將僅替代澳洲產品，對我產業影響亦屬有限。至於紐西蘭蘋果、櫻桃及奇異果等水果可能增加進口，惟該等品項目前已大量進口，增加紐國進口僅是替代其他國家產品，且國內該等產業規模極小，預估影響不大。
問3 紐西蘭承諾開放哪些農產品？對我們有甚麼好處？
答
  1. 紐國承諾在協定生效後，所有農產品關稅均將降為零，估計我目前外銷紐西蘭的農產品中，有七成產品可立即享受零關稅，約可節省26.39萬美元（新臺幣800萬元）的出口成本；主要受惠的出口品項以加工食品為主，預期外銷紐國金額將可增加。 
  2. 本會將充份利用本次協定建立的動植物防疫檢疫平台，協助業者掌握商機，加強臺灣水產品、蛋品、蝴蝶蘭、芒果及荔枝等優質農產品外銷紐西蘭。 
問4 對於鹿茸與液態乳進口，政府有甚麼因應對策？
答
  1. 我國已爭取鹿茸與液態乳採取關稅配額，評估初期對產業不致產生太大影響。 
  2. 在養鹿產業方面，政府將協助業者加強鹿種選育及提升技術，並積極建構鹿茸產地標章認證與加工加值鏈，加強產季電視廣告、觀光區專櫃設點及行銷推廣，以提升國產鹿茸消費形象與附加價值。
  3. 在液態乳方面，則將積極輔導酪農合理化經營，推動參加乳牛群性能改良計畫，強化鮮乳標章認證與消費意識。 
  4. 除了前述因應措施外，倘有產業受到進口影響，本會也將利用「農產品受進口損害救助基金」，採取必要的救助措施。
問5 部分酪農業者擔心鮮乳被保久乳取代，農委會將如何因應？
答
  1. 本會依據國內乳品實際供需情形及可能進口之趨勢，訂定年度國內牛乳生產目標，俾利穩定牛乳年總生產量於32至36萬公噸之間，以穩定產銷及價格。
  2. 本會將輔導國內乳廠與酪農雙方訂定生乳買賣契約，配合生產目標，以落實契約產銷，紓緩季節性剩餘乳壓力，維持國產鮮乳自給率於9成以上。
  3. 持續推廣鮮乳標章，加強認證查核等工作，藉以區隔進口乳品，進而擴大國產乳品市場的占有率。 
  4. 強化國人對乳品消費意識，研發適合國人口味之多樣化乳製品。
  5. 配合農產品生產及驗證管理法之施行，輔導鮮乳業者導入CAS驗證制度，以延續國產優質鮮乳之品牌制度，並確保國產鮮乳品質及衛生安全。
問6 紐西蘭水果會不會大舉來臺？對臺灣果農有何影響？
答
  1. 我國針對蘋果（20%）、櫻桃（7.5%）及奇異果（20%）等水果仍課有進口關稅，依據臺紐ANZTEC，蘋果與櫻桃將立即降為零關稅，至於奇異果則將分3年取消關稅，預期這3種紐西蘭水果將可能增加。
  2. 由於這3種水果目前已大量進口，國內生產比例極低，增加紐西蘭水果進口將僅替代其他國家產品，評估我水果產業之直接影響不大；惟鑒於水果消費有替代性，本會也將針對國內水果產業，建置果樹專區，建立產銷供應鏈，並加強輔導外銷與加工，以提升我水果產業競爭力。
  3. 除了前述因應措施外，倘有產業受到進口影響，本會也將利用「農產品受進口損害救助基金」，採取必要的救助措施。 
問7 紐西蘭水產品進口對我產業有甚麼影響？
答
  1. 針對可能增加進口的淡菜，我國已於談判時爭取採取8年的降稅期程。
  2. 由於水產品消費具有替代性，部分紐西蘭水產品增加進口，可能影響我養殖產業（如牡蠣、文蛤、鱸魚等）。對此，本會除加強建立高效率的產銷體系與協助產品開發外，並鼓勵水產品參加認驗證以提昇產品價值，同時結合地產地銷、低碳足跡等消費概念，加強國產品行銷。
  3. 除了前述因應措施外，倘有產業受到進口影響，本會也將利用「農產品受進口損害救助基金」，採取必要的救助措施。 
問8 政府如何利用臺紐洽簽ANZTEC，加強兩國農業合作？
答
  1. 臺紐農業合作一直在進行中，未來臺紐雙方將透過ANZTEC之平台，強化雙邊之農業投資、農產貿易及動植物防疫檢疫等各領域的合作與交流。
  2. 透過兩國加強農業合作，我國將利用「自由經濟示範區－農業加值」，輔導加工業者利用國產與紐西蘭之農產原料，結合臺灣特有元素加工或再製後外銷，以共同拓展國際市場。 

    3. 另在養鹿產業方面，臺紐養鹿業者已於99年開始進行交流，並於102年10月組團赴紐研習鹿隻結核病防治方法，103年紐方亦組團來台進行學術及疾病防治研討會，針對養鹿產業學術經驗與飼養管理方式等面向相互交流。
問9 在動植物防疫檢疫方面，臺紐ANZTEC有何內容？
答
  1. 除了市場開放之外，臺紐經濟合作協定也針對兩國如何加強動植物防疫檢疫合作，以專章方式加以規範。
  2. 除規範遵守國際組織與科學證據之基本義務外，雙方將建立｢執行協議｣機制，研商同等效力、驗證、發證、進口檢查等實施細節，並建立合作、通知、緊急措施及資訊交換機制，以及成立聯合管理委員會，強化雙方管理合作基礎。
  3. 臺灣與紐西蘭將藉由協定，促進雙方在防疫檢疫方面的諮商及對話，以強化雙方防疫檢疫管理、農產貿易交流與產業利益。
問10 臺紐ANZTEC有關農產品特定原產地規則的內容為何？
答
  1. 為避免第三國產品經由轉運即可享受臺紐ANZTEC的優惠關稅，臺紐雙方已針對各項產品訂定可適用優惠關稅的原產地規則。
  2. 初級農產品（如冷凍冷藏畜禽肉、蔬果等）：多規定須在一方飼養或種植，才能享受優惠關稅。以奇異果為例，必須在紐西蘭種植與收穫，出口到臺灣時才能享有ANZTEC的優惠關稅。
  3. 加工食品（如果汁、調製食品等）：多要求在進口原料加工後，產品必須實質轉型，或規定須加工增值40%以上，才能視為該國原產。以冷凍柳橙汁為例，必須是紐西蘭國產柳橙加工而成的果汁，或是進口柳橙加工後，果汁的價值比原料提高40%以上，才能享受優惠關稅。
