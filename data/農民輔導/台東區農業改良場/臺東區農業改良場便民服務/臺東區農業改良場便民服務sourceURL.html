便民服務

(服務課室：作物環境課)

 

  • 植體及土壤診斷分析：分析及診斷流程圖 pdf、作業須知 pdf、營養診斷申請表 doc odt
  • 植物病蟲害鑑定及診斷：診斷流程圖 pdf、作業須知 pdf、診斷諮詢服務紀錄表 doc odt
  • 技術移轉服務：技術移轉申請流程圖 pdf

 

 

(服務課室：作物改良課、斑鳩分場)

 

  • 農作物栽培管理技術：技術諮詢服務流程圖 pdf、作業須知 pdf、技術諮詢服務申請單 doc odt、技術服務團名冊（聯絡資訊）pdf
  • 作物栽培手冊：以本場出版刊物為主，依作物別（稻作、果樹、園藝蔬花、雜糧、有機、休閒與食譜、其他）提供栽培技術教學資訊，內容有作物栽培、合理化施肥推薦用量、病蟲害防治等相關栽培技術資料。
  • 生物材料索取申請：申請流程圖（含作業須知）pdf、申請單 doc odt

 

 

(服務課室：農業推廣課)

 

  • 農民學院：報名流程圖 pdf、報名須知 pdf、報名步驟說明 pdf、網址：https://academy.coa.gov.tw
  • 農業學習護照：申請流程圖 pdf、申請作業須知 pdf、申請單 doc odt
  • 農業相關課程資訊：農民學院_線上學習
  • 本場出版刊物索取：索取流程圖 pdf、索取須知 pdf、索取申請單（出版刊物 doc odt；技術專刊 doc odt）
  • 來場參訪：申請流程圖（含參訪須知）pdf、申請單 doc odt
  • 單一窗口 (農業相關諮詢服務)： 089-325110 分機801 / 089-338713 / service@mail.ttdares.gov.tw
